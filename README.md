# group_admin - 群管理插件

一个功能完整的 NoneBot2 群管理插件，支持多种群管理操作，且只有管理员或群主才能使用。

## 功能特性

- 🛡️ **权限控制**：所有命令仅管理员/群主可用
- 🤖 **机器人权限检测**：机器人必须有管理员权限才能使用管理命令
- 📋 **成员管理**：踢人、封禁、禁言等
- 👑 **身份管理**：设置管理员、修改名片、专属头衔等
- ⚙️ **群设置**：修改群名、加群方式、群搜索等
- 📊 **信息查询**：群信息、成员列表、成员信息等

## 安装方式

该插件已内置在项目中，无需额外安装。确保在 `bot.py` 中已加载：

```python
nonebot.load_plugin("src.plugins.group_admin")
```

## 使用说明

### 权限要求

1. **机器人权限**：机器人必须是群管理员或群主
2. **用户权限**：命令使用者必须是群管理员或群主

### 可用命令

#### 成员管理

| 命令 | 说明 | 示例 |
|------|------|------|
| `/踢人 <QQ号> [原因]` | 踢出群成员 | `/踢人 123456789 广告` |
| `/封禁 <QQ号> [原因]` | 踢出并拉黑 | `/封禁 123456789 违规` |
| `/解封 <QQ号>` | 解除封禁 | `/解封 123456789` |
| `/禁言 <QQ号> [分钟]` | 禁言成员（默认60分钟） | `/禁言 123456789 30` |
| `/解除禁言 <QQ号>` | 解除禁言 | `/解除禁言 123456789` |
| `/全员禁言` | 开启全员禁言 | `/全员禁言` |
| `/解除全员禁言` | 关闭全员禁言 | `/解除全员禁言` |

#### 身份管理

| 命令 | 说明 | 示例 |
|------|------|------|
| `/设管理员 <QQ号>` | 设置管理员 | `/设管理员 123456789` |
| `/取消管理员 <QQ号>` | 取消管理员 | `/取消管理员 123456789` |
| `/改名片 <QQ号> <新名片>` | 修改群名片 | `/改名片 123456789 新昵称` |
| `/专属头衔 <QQ号> <头衔>` | 设置专属头衔 | `/专属头衔 123456789 VIP会员` |

#### 群设置

| 命令 | 说明 | 示例 |
|------|------|------|
| `/改群名 <新群名>` | 修改群名称 | `/改群名 我的新群` |
| `/群备注 <备注>` | 修改群备注 | `/群备注 游戏交流群` |
| `/加群方式 <允许/拒绝/需验证>` | 设置加群方式 | `/加群方式 需验证` |
| `/群搜索 <开启/关闭>` | 设置群可搜索 | `/群搜索 开启` |

#### 信息查询

| 命令 | 说明 |
|------|------|
| `/群信息` | 查看群详细信息 |
| `/成员列表` | 查看群成员列表 |
| `/成员信息 <QQ号>` | 查看成员详细信息 |

#### 帮助命令

| 命令 | 说明 |
|------|------|
| `/群管理菜单` | 查看所有群管理命令 |
| `/群管` | 快捷命令，同群管理菜单 |

## 配置说明

无需额外配置，插件会自动检测权限。

## 注意事项

1. 机器人必须拥有管理员或群主权限才能执行管理操作
2. 普通成员使用管理命令会收到权限不足提示
3. 如果机器人没有管理员权限，会提示"暂无权限，无法使用管理菜单！"
4. 建议将机器人设置为管理员，以充分发挥插件功能

- NoneBot2
- OneBot V11 适配器
- Python 3.10+
